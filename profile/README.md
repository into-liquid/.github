## 目的

物体の実際の大きさや形状を仮想体験させる手法として、HMD:Head Mounted Display装置を使った3D表現は、一般的な平面2Dモニターを使った3D表現より優れている。
自分で、HMD装置を使った3D表現を行いたいなら、学ばなければならない。

- 3D表示
- 3D表示する物体の作成
- 物体の物理挙動
- 没入表示
- 周囲環境の合成

3D表示、没入表示についての導入は以下のリポジトリを参照。

- [Support-VRBook](https://github.com/into-liquid/Support-VRBook)

その導入方法と簡単な実践を紹介している。
